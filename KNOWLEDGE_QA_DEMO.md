## 🚀 使用演示

### 步骤1：启动功能
1. 打开智能助手页面
2. 点击输入框下方的 **"知识问答"** 按钮（🔵❓图标）
3. 系统显示欢迎消息和功能介绍
4. 输入框占位符变为问答提示

### 步骤2：提问测试
尝试以下不同类型的问题：

#### 🔬 概念解释类
```
问：什么是机器学习？
答：详细解释机器学习的定义、特点、分类和应用领域
```

#### 🛠️ 方法指导类  
```
问：如何学习Python编程？
答：提供完整的学习路径、步骤和实用建议
```

#### 💡 推荐建议类
```  
问：推荐几个好用的代码编辑器
答：对比分析不同编辑器的优缺点和适用场景
```

#### 🔍 综合问答类
```
问：深度学习和机器学习有什么区别？
答：详细比较两者的关系、差异和应用
```

### 步骤3：查看回答
系统会分两部分显示回答：

1. **🤔 思考过程**
   - 问题分析和解决思路
   - 回答的逻辑结构
   - 考虑因素说明

2. **💡 最终回答**
   - 结构化的详细解答
   - 相关背景和应用
   - 实用建议和注意事项
   - Token使用统计

## 📊 技术特性

### 智能问答引擎
- **问题分析**：自动识别问题类型和意图
- **针对性回答**：根据不同类型提供专业回答
- **推理展示**：模拟星火大模型的思考过程
- **格式优化**：结构化的回答展示

### 参数配置
- **温度控制**：调节回答的创造性（0.1-2.0）
- **Token限制**：控制回答长度（1-131072）
- **网络搜索**：启用实时信息增强
- **会话追踪**：每次问答生成唯一ID

### API接口规范
