# 智能助手页面修改测试指南

## 测试环境
- URL: http://127.0.0.1:5000/intelligent-assistant/
- 浏览器: Chrome/Firefox/Edge (推荐使用最新版本)

## 测试步骤

### 1. 基础功能测试

#### 1.1 页面加载测试
- [ ] 访问 http://127.0.0.1:5000/intelligent-assistant/
- [ ] 确认页面正常加载
- [ ] 确认输入框下方显示两个按钮：
  - PPT 生成
  - 思维导图生成
- [ ] 确认按钮样式正确（圆角、图标、文字）

#### 1.2 按钮样式测试
- [ ] 鼠标悬停在"PPT 生成"按钮上
  - 确认边框变为蓝色
  - 确认按钮轻微上移
  - 确认出现阴影效果
- [ ] 鼠标悬停在"思维导图生成"按钮上
  - 确认同样的悬停效果
- [ ] 鼠标移开后按钮恢复原状

### 2. PPT 生成功能测试

#### 2.1 点击按钮测试
- [ ] 点击"PPT 生成"按钮
- [ ] 确认欢迎界面消失
- [ ] 确认AI助手显示引导消息
- [ ] 确认引导消息包含：

  - 提示可以输入简单科目名称
- [ ] 确认输入框占位符变为"输入PPT生成要求，例如：机器学习"
- [ ] 确认输入框自动获得焦点

#### 2.2 输入和发送测试
- [ ] 在输入框输入"机器学习"
- [ ] 按Enter键发送
- [ ] 确认消息正常发送到聊天区域
- [ ] 确认AI开始处理请求（显示"思考中"动画）

#### 2.3 复杂输入测试
