# 快速启动指南

## 查看修改效果

### 1. 启动应用

在项目根目录下运行：

```bash
# Windows
python run.py

# 或使用批处理文件
start.bat
```

### 2. 访问页面

在浏览器中打开：
```
http://127.0.0.1:5000/intelligent-assistant/
```

### 3. 查看新按钮

页面加载后，你会看到：

```
┌──────────────────────────────────────┐
│  [输入框]                             │
│  输入您的问题...                      │
└──────────────────────────────────────┘
─────────────────────────────────────────
  [📄 PPT 生成]  [🗺️ 思维导图生成]
```

### 4. 测试PPT生成

1. 点击 **[PPT 生成]** 按钮
2. 看到AI助手的引导消息
3. 在输入框输入："机器学习"
4. 按 **Enter** 发送
5. AI会以对话形式处理你的请求

### 5. 测试思维导图生成

1. 点击页面右上角的 **[清空]** 按钮
2. 点击 **[思维导图生成]** 按钮
3. 看到AI助手的引导消息
4. 在输入框输入："数据结构"
5. 按 **Enter** 发送
6. AI会生成思维导图

## 修改的文件

只修改了一个文件：
```
1/app/templates/intelligent_assistant/index.html
```

修改内容包括：
- ✅ 添加了CSS样式（`.resource-action-buttons` 和 `.resource-action-btn`）
- ✅ 添加了两个按钮的HTML
- ✅ 添加了两个JavaScript函数（`openPptDialog()` 和 `openMindmapDialog()`）

## 关键特性

### 按钮外观
- **圆角设计**：border-radius: 20px
- **悬停效果**：
  - 边框变蓝
  - 轻微上移
  - 添加阴影
- **图标**：
  - PPT: fa-file-powerpoint
  - 思维导图: fa-sitemap

### 对话模式
用户点击按钮后：
1. 欢迎界面消失
2. AI显示引导消息
3. 输入框占位符更新
4. 自动聚焦输入框
5. 用户输入后正常发送

### 智能引导
AI会告诉用户：
- 如何使用这个功能
- 提供具体的输入示例
- 说明会返回什么结果

## 如果遇到问题

### 按钮没有显示
1. 检查浏览器控制台是否有JavaScript错误
2. 刷新页面 (Ctrl+F5 强制刷新)
3. 清除浏览器缓存

### 按钮样式不正确
1. 检查CSS是否正确加载
2. 尝试在不同浏览器中测试
3. 检查浏览器开发者工具的样式面板

### 点击按钮无反应
1. 打开浏览器控制台查看错误
2. 确认JavaScript函数已正确加载
3. 检查是否有其他脚本冲突
