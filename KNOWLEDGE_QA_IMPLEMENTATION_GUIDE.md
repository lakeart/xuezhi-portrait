# 知识问答功能实现指南

## 🎯 功能概述

基于讯飞星火大模型的智能知识问答功能已成功集成到智能助手系统中。当前实现包含完整的前后端交互，支持多种类型的智能问答。

## ✅ 已实现功能

### 🖥️ 前端功能
1. **知识问答按钮** - 位于资源生成按钮区域
2. **对话模式** - 点击按钮后进入专门的知识问答模式
3. **智能提示** - 自动更新输入框占位符
4. **实时反馈** - 显示推理过程和最终答案
5. **错误处理** - 完善的错误提示和重试机制

### 🔧 后端API
1. **接口路由** - `/agent/knowledge/ask` (POST)
2. **参数配置** - 支持温度、Token数量、网络搜索等参数
3. **智能回答** - 根据问题类型提供针对性回答
4. **使用统计** - Token使用情况和会话ID追踪
5. **错误处理** - 详细的错误信息和调试日志

## 🚀 使用方法

### 启动知识问答
1. 打开智能助手页面
2. 点击 **"知识问答"** 按钮（蓝色问号图标）
3. 系统显示引导消息和功能说明
4. 输入框占位符更新为问答提示

### 提问方式
支持多种类型的问题：

**📚 概念解释类**
```
- 什么是机器学习？
- 人工智能的定义是什么？
- 解释一下区块链技术
```

**🛠️ 方法指导类**
```
- 如何学习Python编程？
- 怎样提高编程能力？
- 如何准备技术面试？
```

**💡 推荐建议类**
```
- 推荐几个好用的代码编辑器
- 建议学习哪些前端框架？
- 选择什么数据库比较好？
```

**🔍 综合问答类**
```
- 深度学习和机器学习有什么区别？
- 微服务架构有什么优缺点？
- 前端开发的发展趋势如何？
```

### 回答展示
系统会分两部分显示回答：

1. **🤔 思考过程**（推理内容）
   - 分析问题的思路
   - 解决方案的考虑因素
   - 回答的逻辑结构

2. **💡 最终答案**（详细回答）
   - 结构化的详细解答
   - 相关背景和应用
   - 实用建议和注意事项
   - Token使用统计信息

## 🔧 技术实现

### API接口详情

**请求地址：** `POST /agent/knowledge/ask`

**请求参数：**
```json
{
    "question": "您的问题",
    "enable_search": true,      // 可选，是否启用网络搜索
    "temperature": 0.7,         // 可选，回答创造性 (0.1-2.0)
    "max_tokens": 4096          // 可选，最大回答长度 (1-131072)
}
```

**响应格式：**
```json
{
    "ok": true,
    "content": "详细回答内容",
    "reasoning_content": "推理过程",
    "usage": {
        "total_tokens": 1234,
        "prompt_tokens": 567,
        "completion_tokens": 667,
        "search_prompt_tokens": 20
    },
    "sid": "会话ID"
}
```

### 升级到真实API
要升级到真实的星火WebSocket API，需要：

1. **安装WebSocket依赖**
   ```bash
   pip install websocket-client
   ```

2. **实现WebSocket客户端**
   - 按照星火API文档实现WebSocket连接
   - 处理流式消息接收和解析
   - 实现鉴权和错误处理

3. **替换模拟函数**
   - 将 `_call_spark_knowledge_api` 替换为真实的WebSocket实现
   - 保持API接口不变，确保前端兼容性

## 🛠️ 故障排除

### 常见问题

1. **"缺少WebSocket模块"**
   - 当前使用智能模拟版本，不需要WebSocket
   - 如需真实API，安装：`pip install websocket-client`

2. **"API返回错误"**
   - 检查问题是否为空
   - 确认参数格式正确
   - 查看服务器日志获取详细信息

3. **"响应时间过长"**
   - 模拟版本响应很快，真实API可能较慢
   - 检查网络连接和服务器负载

4. **"回答质量不满意"**
   - 当前为演示版本，真实API质量更高
   - 可调整temperature参数控制创造性
   - 尝试更具体明确的问题表述
