# 计算机组成原理知识库样例包

本目录用于作为 A3 赛题提交附件中的“完整课程知识库样例包”。

## 课程定位

- 课程名称：计算机组成原理
- 适用对象：计算机类专业本科生
- 样例用途：
  - 上传到系统知识库模块，验证 TXT / Markdown / CSV / JSON 等材料的解析与索引能力
  - 用于 RAG 问答演示
  - 用于资源生成、学习计划推荐和能力补弱场景说明

## 文件组成

1. `01_课程导学.md`
   - 课程目标、章节安排、学习建议
2. `02_核心知识讲义.md`
   - 冯诺依曼结构、指令执行、流水线、Cache、中断等核心内容
3. `03_实验指导.md`
   - 面向实验课的任务说明与常见错误提醒
4. `04_章节题库.csv`
   - 章节习题与标准答案样例
5. `05_术语表.json`
   - 课程核心术语与解释
6. `06_复习提纲.txt`
   - 面向期中 / 期末复习的压缩提纲

## 推荐演示问题

1. 什么是冯诺依曼结构？
2. Cache 的主要作用是什么？
3. 指令周期一般包括哪几个阶段？
4. 流水线为什么会出现冒险？
5. 中断和程序查询方式有什么差别？
